N\+T\+B++ is a header-\/only utility library for C++ $>$= 17.

It includes Colors, Templates \& Types. Useful for printing colors, {\ttfamily std\+::map} and {\ttfamily std\+::vector} into {\ttfamily std\+::cout}.~\newline
 A couple of common typealiases (typedefs) like {\ttfamily String}, {\ttfamily U\+Int}, {\ttfamily String\+Map}, {\ttfamily Map\+Of$<$Type$>$} are included as well.~\newline


Motivation\+:

We developed this lib as part of a program for a customer,~\newline
 and we figured it might benefit other developers too.
